## Negotiation Workflow

Parley's negotiation tools enable collaborative contract discussions with AI-powered insights. Both parties can suggest changes, comment on clauses, and track progress in real-time.

<Steps>
  <Step title="Send for Review" icon="send">
    Share the contract draft with all parties via email or direct link.
  </Step>
  <Step title="Review Suggestions" icon="eye">
    AI highlights potential issues and suggests improvements.
  </Step>
  <Step title="Discuss Changes" icon="message-square">
    Use in-line comments and negotiation threads for discussion.
  </Step>
  <Step title="Accept & Sign" icon="check-circle">
    Finalize terms and proceed to electronic signatures.
  </Step>
</Steps>

<Tabs>
  <Tab title="Redline Mode" icon="edit">
    ```javascript
const negotiation = await parley.startNegotiation(contractId, {
  mode: 'redline',
  participants: ['party1@example.com', 'party2@example.com']
});
```
  </Tab>
  <Tab title="Collaborative Editing" icon="users">
    ```javascript
const session = await parley.createNegotiationSession(contractId, {
  allowComments: true,
  requireApproval: false
});
```
  </Tab>
</Tabs>

<Callout kind="success">Negotiation sessions can reduce contract cycle time by up to 50% through streamlined communication.</Callout>
